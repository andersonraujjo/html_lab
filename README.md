# 🌐 Exercícios de HTML5 e CSS3 - Trilha de Aprendizado

Este repositório é um registro prático da minha evolução no desenvolvimento de interfaces **Front-end**. O foco aqui é a transição da lógica pura (Back-end) para a construção visual, estruturando páginas semânticas e estilizações modernas.

## 📈 Evolução Técnica e Estruturação

Como este projeto acompanha as aulas teóricas e práticas da faculdade e cursos complementares (como a DIO), o repositório está organizado para refletir o amadurecimento no design e código:

* **Fase 01 (Fundamentos):** Foco total em **HTML5 Semântico**. O objetivo aqui foi entender a hierarquia das tags (`header`, `main`, `section`, `footer`) antes de focar na estética.
* **Fase 02 (Estilização):** Introdução ao **CSS3**, trabalhando com seletores, box model e a importância da separação entre estrutura (HTML) e estilo (CSS).
* **Fase 03 (Layouts):** Exercícios focados em **Flexbox** e **Responsividade**, garantindo que as páginas se adaptem a diferentes tamanhos de tela.

## 🗂️ O que foi praticado:

* **Estruturação HTML:** Tags de texto, parágrafos, títulos, links internos/externos e listas (ordenadas e não ordenadas).
* **Mídias:** Incorporação correta de imagens e vinculação de arquivos de estilo.
* **CSS Básico:** Manipulação de cores, fontes, margens e o conceito de *cascata*.
* **Organização de Arquivos:** Estrutura de pastas padronizada para facilitar a manutenção e o estudo.

## 🛠️ Tecnologias Utilizadas

* **Linguagens:** HTML5 e CSS3
* **Ambiente de Dev:** VS Code (Extensão Live Server)
* **Versionamento:** Git & GitHub
